# 声形 Koe-Katachi

声のかたち(声調の輪郭)でピンインを覚える学習用Webアプリです。

https://jacqueline37.github.io/koekatachi/

## できること

- **声調トレーナー**: 発音を聞いて、4つの声調のかたちから正解を選ぶクイズ
- **単語カード**: ライトナー式の間隔反復(SRS)で語彙を復習
- **発音練習**: お手本を聞いて自分の声を録音し、聞き比べる

PC(左サイドバー表示)・スマホ(上部タブ表示)どちらの画面サイズにも対応しています。

## データの扱いについて

- 学習の進捗はこのブラウザの中(localStorage)だけに保存されます。サーバーには送信されません。
- 発音練習で録音した声は、その場で聞き比べるためだけにブラウザのメモリ上で再生され、どこにも送信・保存されません。次の単語に進む、またはタブを閉じると自動的に破棄されます。
